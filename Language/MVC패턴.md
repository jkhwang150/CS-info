# MVC 패턴
- 모델(model)-뷰(view)-컨트롤러(controller)으로 구성되어 있으며 주로 사용자 인터페이스, 데이터 및 논리 제어를 구현하는데 사용되는 디자인 패턴이다.
- 장고의 대표적인 디자인 패턴이다.
#### 모델(model) : 데이터와 비즈니스 로직을 관리한다.
#### 뷰(view) : 레이아웃과 화면을 처리한다.
#### 컨트롤러(controller) : 모델과 뷰로 명령을 전달한다.
